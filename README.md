🚀 Day 3 – RAG Bootcamp

This project is part of the **Generative & Agentic AI Bootcamp** conducted by Nunnari Academy.

 📌 Overview
In this task, I implemented a **Retrieval-Augmented Generation (RAG)** pipeline using LangChain and ChromaDB.

🔍 Key Features
-  Loaded multiple PDF documents
-  Split documents into smaller chunks
-  Generated embeddings using HuggingFace
-  Stored embeddings in ChromaDB (Vector Database)
-  Built a semantic retriever (Top-K search)
-  Implemented a RAG pipeline
-  Tested with multiple questions

⚙️ Technologies Used
- Python
- LangChain
- ChromaDB
- HuggingFace Transformers

🧠 How It Works
1. Load PDF documents  
2. Split into chunks  
3. Convert chunks into embeddings  
4. Store in vector database  
5. Retrieve relevant chunks for a query  
6. Generate answer using LLM  

📸 Output
The system retrieves relevant chunks from documents and generates answers based on context.
